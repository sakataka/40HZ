# 40Hz Stimulation

A React/Vite app that generates and plays 40 Hz isochronic tones in the browser.

## Scripts

- `bun install`
- `bun run dev`
- `bun run build`
- `bun test`

## Notes

- This is not a medical device.
- Playback is locked until the safety information is acknowledged on first use.
- Some mobile browsers may not keep playback stable while the screen is locked.
